**Zalith Launcher 2** is a newly designed launcher for **Android devices** tailored for [Minecraft: Java Edition](https://www.minecraft.net/). The project utilizes [PojavLauncher](https://github.com/PojavLauncherTeam/PojavLauncher/tree/v3_openjdk/app_pojavlauncher/src/main/jni) as its core launching engine, featuring a modern UI built with **Jetpack Compose** and **Material Design 3**.

::: warning Reminder
This project is entirely separate from [ZalithLauncher](/en/docs/projects/zl1). The project is currently in development, with features being added regularly. Feel free to follow updates on [Github](https://github.com/ZalithLauncher/ZalithLauncher2)!
